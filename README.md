# Solar Management System (PHP)

Este projeto consiste em um sistema web desenvolvido durante o estágio supervisionado
realizado na empresa Norte Sol Energia Solar Matriz, com o objetivo de auxiliar no
gerenciamento de clientes e organização de informações comerciais.

## Contexto
O sistema foi desenvolvido a partir da identificação da necessidade da empresa por uma
plataforma própria, mais simples e adequada ao seu fluxo interno, substituindo o uso de
múltiplos sistemas externos.

## Funcionalidades
- Cadastro de clientes
- Listagem, atualização e remoção de registros (CRUD)
- Pesquisa por nome
- Paginação e listagem dinâmica
- Integração com banco de dados relacional

## Tecnologias Utilizadas
- PHP
- HTML
- CSS
- JavaScript
- Banco de dados relacional (MySQL)

## Aprendizados
O projeto proporcionou experiência prática com desenvolvimento web, modelagem de banco
de dados e compreensão de processos reais de desenvolvimento de software em ambiente
corporativo.
